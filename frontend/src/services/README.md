# Services

This directory contains logic for interacting with external APIs and backend services.

## Contents

- `api.ts`: Base Axios configuration and interceptors.
- `authService.ts`: Methods for login, logout, and token management.
- `dashboard.service.ts`: Fetches data for the dashboard views.
- `attachmentService.ts`: Handles file attachments and uploads.
- `dependencyService.ts`: Manages task dependencies.
- `metricsService.ts`: Fetches project and team metrics.
- `notificationService.ts`: Manages user notifications.
- `webhookService.ts`: Handles configuration of external webhooks.
