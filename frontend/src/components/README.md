# Components

This directory contains reusable UI components used throughout the application.

## Main Components

- `CardModal.tsx`: Modal for displaying and editing task details.
- `CreateTaskModal.tsx`: Modal for creating new tasks.
- `DraggableCard.tsx`: Draggable task card used in the board.
- `DroppableColumn.tsx`: Drop zone for task cards in the board columns.
- `FileUploader.tsx`: Component for handling file uploads.
- `MetricsDashboard.tsx`: Visualization for project metrics.
- `NotificationBadge.tsx`: Badge for showing unread notifications.
- `Sidebar.tsx`: Main navigation sidebar.
- `TeamModal.tsx`: Modal for managing team members.
- `WebhookSettings.tsx`: Interface for configuring webhooks.
- `WorkloadView.tsx`: Visualization for member workload.

## Subdirectories

- `auth/`: Specialized components for authentication flows.
