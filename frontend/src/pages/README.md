# Pages

This directory contains page-level components that correspond to specific routes in the application.

## Contents

- `BoardPage.tsx`: The main Kanban board interface.
- `DashboardPage.tsx`: Overview page with metrics and summaries.
- `LoginPage.tsx`: User authentication page.
- `RegisterPage.tsx`: User account creation page.
- `ProfilePage.tsx`: User profile and settings management.
- `ProjectsPage.tsx`: List and management of user projects.
- `TeamPage.tsx`: Management of team members and roles.
- `NotificationsPage.tsx`: View for user notifications.
- `ForgotPasswordPage.tsx`: Initiation of password recovery.
- `ResetPasswordPage.tsx`: Finalizing password reset.
- `SprintReportPage.tsx`: Generated reports for sprints.
- `PublicReportPage.tsx`: Viewable reports for external stakeholders.
